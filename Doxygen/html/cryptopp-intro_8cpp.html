void IntegerExercise ( )

Funktion vom Datentyp void um den Restwert der modularen Exponentiation drei Integer auszugeben.

Funktion vom Datentyp void die Anfangs zwei Konsolenausgaben betätigt um bessere lesbarkeit der
Konsole zu erreichen.Die Integer x("371"), Integer e("18961551"), Integer m("2371"), Integer t = modexp(x, e, m) und
Integer r = a_exp_b_mod_c(x, e, m) werden initialisiert. Den Funktionen modeexp und a_exp_b_mod_c werden
die selben Parametern übergeben, um ein Ergebnis zum Vergleich zu haben ,da sie den selben Zweck erfüllen sollen.

Grundform: a = y ^ 2 % p == a = y * y % p Diese zwei Gleichungen sind identisch.
eulersche Kriterium: a ^ ((p - 1) / 2) % p
zu y = (p - 1) / 2 eingesetzt in a ^ y % p Teil des eulerschen Kriteriums zu einer Variablen geformt.
d = a ^ y % p == modexp(a,y,p) = d Die Rechnung der Gleichung wird in der Funktion gegeben.

Die Integer a("58788"), Integer p("95957"), Integer y = (p - 1) / 2 und Integer q = modexp(a, y, p) werden initialisiert.
Anschließend werden t, r und q untereinander in der Konsole ausgegeben.
